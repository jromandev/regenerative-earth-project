# CONTRIBUTING.md

## Contributing to The Regenerative Earth Project

Thank you for your interest in contributing. This project exists to support regenerative, community-centered development around the world. Contributions are welcome from developers, researchers, engineers, designers, and practitioners.

Our goal is to build a transparent and ethical platform that helps communities develop infrastructure using locally available natural resources.

---

## Ways You Can Contribute

### Code
Help develop the platform, APIs, and tools that analyze geographic and ecological data.

### Research
Contribute knowledge related to:

- Soil and hydrology
- Permaculture and regenerative agriculture
- Natural building techniques
- Renewable energy systems
- Indigenous land stewardship practices

### Documentation
Clear documentation ensures the project remains accessible to contributors worldwide.

### Data Integration
Help integrate open environmental datasets such as climate, terrain, and biodiversity data.

### Field Knowledge
Real-world experience from engineers, humanitarian workers, farmers, builders, and ecologists is extremely valuable.

---

## Contribution Principles

All contributions should align with the core principles of the project:

- Ecological regeneration
- Community sovereignty
- Transparency
- Non-exploitation
- Local feasibility
- Safety

Any contribution that promotes harmful environmental practices, militarization, or exploitative resource extraction will not be accepted.

---

## Contribution Workflow

1. Fork the repository
2. Create a new branch for your feature or improvement
3. Commit your changes with clear descriptions
4. Submit a pull request
5. Participate in review and discussion

---

## Communication

Constructive discussion and respectful collaboration are essential to this project.

We welcome thoughtful debate and interdisciplinary perspectives.

---

## A Note on Stewardship

This project operates under a stewardship model. Contributors help shape the system, but the mission and ethical guardrails must always remain intact.

Thank you for helping build a regenerative future.
